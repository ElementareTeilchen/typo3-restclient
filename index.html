Typo3-restclient

A REST client for TYPO3 CMS

Download as .zip Download as .tar.gz View on GitHub

Introduction

This extension basically provides a class library for sending HTTP requests and receiving HTTP responses from a RESTful API resource.
It supports the CRUD operations with the shorthands get/post/put/delete.
You can set many options such as logs, timeout, proxy with basic/digest authentication, ssl certificate, cookies, and others.
The HttpClient class is cURL library based.

Requirements

It requires TYPO3 6.2 and PHP 5.3.
Also it uses cURL librabry, then you should have installed the libcurl package (>=7.10.5).
